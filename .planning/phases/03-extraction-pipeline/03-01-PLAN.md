```python
def get_transcript_text(cache_path: Path) -> str:
    """Extract plain transcript string from cached Deepgram JSON."""
    data = json.loads(cache_path.read_text(encoding="utf-8"))
    return data["results"]["channels"][0]["alternatives"][0]["transcript"]
```

Transcript JSON structure:
```json
{"metadata": {...}, "results": {"channels": [{"alternatives": [{"transcript": "Guten Tag...", "words": [...]}]}]}}
```

```python
    ),

phone_number: Optional[str] = Field(
```

**transcribe_node** per D-07: async function that loads cached transcript from `data/transcripts/{recording_id}.json` via `get_transcript_text()`. If cache file does not exist, raise `FileNotFoundError` with the path (Phase 2 ensures all 30 are cached; API fallback is not needed in Phase 3). Returns `{"transcript_text": text}`.

**extract_node** per D-08, D-10, D-11: async function that:
1. Creates `ChatAnthropic(model=os.getenv("PHONEBOT_MODEL", "claude-sonnet-4-6"))` -- model name from env var, defaulting to claude-sonnet-4-6
2. Calls `model.with_structured_output(CallerInfo, method="json_schema")` to get structured output model
3. Calls `await structured_model.ainvoke(state["transcript_text"])` -- passes the full transcript text (D-04)
4. Returns `{"caller_info": result.model_dump()}` -- converts CallerInfo to dict to avoid Pitfall 1

**build_pipeline** per D-06: function that constructs StateGraph with topology `START -> "transcribe" -> "extract" -> END` and returns `builder.compile()`.

**PIPELINE** module-level constant: `PIPELINE = build_pipeline()` -- built once at import time per anti-pattern guidance.

**run_pipeline** per D-09: async function with signature:
```python
async def run_pipeline(
    recording_ids: list[str],
    model_name: str = "claude-sonnet-4-6",
    concurrency: int = 5,
) -> list[dict]:
```
- Sets `os.environ["PHONEBOT_MODEL"] = model_name` so extract_node picks it up
- Creates `asyncio.Semaphore(concurrency)` bounded by `EXTRACT_CONCURRENCY` env var (default 5)
- For each recording_id, creates a task that:
  - Acquires semaphore
  - Calls `await PIPELINE.ainvoke({"recording_id": recording_id, "transcript_text": None, "caller_info": None})`
  - Returns `{"id": recording_id, "caller_info": state["caller_info"], "model": model_name, "timestamp": datetime.now(timezone.utc).isoformat()}` per D-12
- Uses `asyncio.gather(*tasks)` to run concurrently
- Returns `list(results)`
