    **Step 4: Create transcription module (GREEN phase)**
    Create `src/phonebot/pipeline/transcribe.py` with exactly these public functions:

    - `_transcribe_one(client: AsyncDeepgramClient, wav_path: Path, semaphore: asyncio.Semaphore) -> tuple[str, Path]`:
      - `recording_id = wav_path.stem`
      - `cache_path = TRANSCRIPT_DIR / f"{recording_id}.json"`
      - If `cache_path.exists()`: return `(recording_id, cache_path)` immediately (per D-02)
      - Otherwise: create `PrerecordedOptions(model="nova-3", language="de", smart_format=True, punctuate=True, diarize=True, paragraphs=True)` (per D-03)
      - `async with semaphore`: read wav bytes, call `await client.listen.v1.media.transcribe_file(request=audio_bytes, options=options)`
      - `TRANSCRIPT_DIR.mkdir(parents=True, exist_ok=True)`
      - `cache_path.write_text(response.to_json(), encoding="utf-8")` (per D-01)
      - Return `(recording_id, cache_path)`

    - `transcribe_all(recordings_dir: Path = Path("data/recordings")) -> dict[str, Path]`:
      - `client = AsyncDeepgramClient()` (reads DEEPGRAM_API_KEY from env)
      - `semaphore = asyncio.Semaphore(CONCURRENCY)` where `CONCURRENCY = int(os.getenv("DEEPGRAM_CONCURRENCY", "5"))` (per D-07)
      - `wav_files = sorted(recordings_dir.glob("call_*.wav"))`
      - `results = await asyncio.gather(*[_transcribe_one(client, wav, semaphore) for wav in wav_files])`
      - Return `dict(results)`

    - `get_transcript_text(cache_path: Path) -> str`:
      - `data = json.loads(cache_path.read_text(encoding="utf-8"))`
      - Return `data["results"]["channels"][0]["alternatives"][0]["transcript"]`

    - `get_words(cache_path: Path) -> list[dict]`:
      - `data = json.loads(cache_path.read_text(encoding="utf-8"))`
      - Return `data["results"]["channels"][0]["alternatives"][0]["words"]`

    Module-level constants:
    - `TRANSCRIPT_DIR = Path("data/transcripts")`
    - `CONCURRENCY = int(os.getenv("DEEPGRAM_CONCURRENCY", "5"))`

    Imports: `asyncio`, `json`, `os`, `Path` from pathlib, `AsyncDeepgramClient` and `PrerecordedOptions` from deepgram, `load_dotenv` from dotenv. Call `load_dotenv()` at module top.
